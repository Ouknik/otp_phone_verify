# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-02

### Added
- Initial release
- Beautiful OTP verification popup dialog
- Multi-language support (English, Arabic, French, Spanish)
- Multiple theme presets (Default, Dark, Blue, Green, Purple)
- RTL (Right-to-Left) support for Arabic
- Customizable translations and themes
- Auto-resend with countdown timer
- Haptic feedback support
- Responsive design for all screen sizes
- Secure API key/secret authentication
- Comprehensive documentation and examples
