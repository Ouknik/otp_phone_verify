# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2026-01-02

### Added
- Complete API documentation link
- Improved README with examples
- Better error handling

### Changed
- Updated package description for pub.dev

## [1.0.2] - 2026-01-02

### Added
- Local OTP verification for faster response
- Server returns OTP code for client-side verification

### Fixed
- Request ID passing in verification flow

## [1.0.1] - 2026-01-02

### Fixed
- Request ID parameter handling in OTP verification

## [1.0.0] - 2026-01-02

### Added
- Initial release
- Beautiful OTP verification popup dialog
- Multi-language support (English, Arabic, French, Spanish)
- Multiple theme presets (Default, Dark, Blue, Green, Purple)
- RTL (Right-to-Left) support for Arabic
- Customizable translations and themes
- Auto-resend with countdown timer
- Haptic feedback support
- Responsive design for all screen sizes
- Secure API key/secret authentication
- WhatsOTP.me integration
- Comprehensive documentation and examples
